# About the Project
This is a customized clone of the classic snake game. The game has google authentication for saving the high score and displaying a leaderboard. The web application is a single-page serverless application. It has been written in the functional programming language Elm along with Firebase support.

# Tech Stack
- Front-end - Elm
- Database - Cloud Firestore
- Authentication - Firebase Authentication
- Hosting Service - Firebase Hosting

# Installing Packages
    > npm install

# Production Build
    > npm start
    
# Deployment Build
    > npm run-script build
